About

Getting over it - but is it? is a project I made for the 2021 CGX Grad Jam - Game Jam, the whole project was done by myself. I used the unity free asset for the level design and online soundtracks for this project, the rest was hand drawed by me

What I've leaned

Photoshop - I learned many PS features like: create shape, layers, time line, crop layer, color overlay and so on...

Unity - I learned how to use new 2D animation, 2D IK, create cutscenes, scene transition animation

Game Desing - I learn how to design level, story, draw out a plan, improved my time management skill

What I've Overcome

From this jam I got to experience a fast pace working evironment. This was my first game jam and the theme was released late so I was working under pressure. Even though the time wasn't strict I still manage to submit my game before the official deadline.
